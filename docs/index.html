Course description

Name Machine Learning Engineering
Intended audience M. Sc. students in Computer Science interested in Data Science and ML
Prerequisites extended knowledge of Python and scikit-learn
basic knowledge of Linux and TCP/IP networks
familiarity with software engineering best practices (e.g. TDD, agile, microservices)
Educational goals familiarity and proficiency in development and operations of ML applications
ability to work as full-stack ML engineers in close collaboration
with software developers and operators
Theory DevOps methodology
cloud technologies and methodologies
(e.g. infrastructure management, compute and storage resources)
machine learning engineering (e.g. build and deployment, quality assurance, REST APIs)
Practice development of a cloud-based sales price prediction service
for the fictional King County Real Estate Company
