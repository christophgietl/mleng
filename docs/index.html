Course description

Name Machine Learning Operations
Intended audience Students of a fictional M. Sc. programme in Data Science
interested in pursuing a career in Data Engineering
Prerequisites extended knowledge of Python and scikit-learn
basic knowledge of Linux and TCP/IP networks
familiarity with software engineering best practices (e.g. TDD, agile, microservices)
Educational goals Students gain familiarity and proficiency in development and operations
of machine learning applications.
Students can work as full-stack data engineers
in close collaboration with software developers and operators.
Theory DevOps methodology
cloud technologies
machine learning operations
Practice Students develop a cloud stack for the fictional King County Real Estate Company
including a sales price prediction service.
