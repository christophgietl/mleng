About this course

  • Intended audience
    • Master students
      • in a fictional Data Science programme
      • who are interested in pursuing Data Engineering career
  • Prerequisites
    • basic knowledge of Linux and TCP/IP networks
      • as taught in most B.Sc. programmes in Computer Science or Data Science
    • extended knowledge of Python and scikit-learn
      • as taught in the fictional course Advanced Data Science Pipelines of the B.Sc. programme Data Science & Scientific Computing
    • familiarity with software engineering best practices
      • test-driven development
      • agile methodologies
      • microservice architectures
  • Educational goals
    • Students gain familiarity and proficiency
      • in deployment of machine learning applications and
      • in operations of machine learning applications.
    • Students in their future career can
      • work as full-stack data engineers and
      • develop and operate machine learning services
        • in cloud environments and in on-premise environments and
        • in collaboration with software developers and software operators while
        • appreciating DevOps technologies and DevOps culture.
  • Content
    • DevOps methodology
    • cloud technologies
    • machine learning operations
